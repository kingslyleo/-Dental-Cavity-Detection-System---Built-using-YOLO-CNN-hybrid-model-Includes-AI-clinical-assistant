# -Dental-Cavity-Detection-System---Built-using-YOLO-CNN-hybrid-model-Includes-AI-clinical-assistant

🦷 Hybrid YOLO + CNN Dental Cavity Detection with AI Clinical Assistant

An end-to-end AI system designed to detect dental cavities from images and assist with clinical insights using a hybrid deep learning approach.

🚀 Key Highlights
Combines YOLO (object detection) + CNN (classification)
Detects cavity regions in dental images
Provides AI-based clinical suggestions
Designed for real-world dental diagnostic support

🧠 How It Works
Input Image → User uploads dental image
YOLO Model → Detects cavity regions
CNN Model → Classifies severity
AI Assistant → Generates clinical insights

🔧 Tech Stack
Python
YOLO (Ultralytics)
Convolutional Neural Networks (CNN)
OpenCV
NumPy / Pandas
Gradio (for UI)
